# Streamlit Portofolio
This repository contains my personal projects to prepare for global internship programs. Each app uses a variey of Python functions, focusing on data science and web applications.

## Project Roadmap
- **App 1: Data Visualization Dashboard**(Pandas, Plotly, Streamlit)
- **App 2: Machine Learning Model Demo**(scikit-learn, MNIST/Iris)
- **App 3: Smart Recommendation Engine**(Search Logoc, CSV/JSON)

## Projects Overview
### 1. Data Visualization Dashboard
**Feature:** Upload CSV data and generate charts and graphs automatically.\
**Tech Stack:** Pandas, Plotly, Streamlit\
**Goal:** Organizing the original data in a intuitive way.\
### 2. Machine Learning Demo
**Feature:** Interactive web UI to try machine learning models.\
**Tech Stack:** scikit-learn, TensorFlow\
**Goal:** Displaying the ability to implement AI in web environments.\
### 3. "Recommend For You" App
**Feature:** A recommendation engine for some categories based on user input.\
**Tech Stack:** Python conditional logic, Ranking algrithums\
**Goal:** Implementing the business logic and database handling.\

## How to Run
Follow these steps to launch the app in GitHub Codespaces.
1. Open in **GitHub Codespaces**.
2. Install Dependencies
   Run the following command in the terminal to install required libraries:
     ~~~bash
     pip install streamlit pandas plotly scikit-learn
3. Start the App:
   Execute the Streamlit command:
     ~~~bash
     streamlit run app.py
4. View the App:
   Click the "Open in Browser" button that appears in the bottom-right corner.
